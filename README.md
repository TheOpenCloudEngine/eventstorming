# eventstorming
Event Storming Wiki Page

## 작성방법

1. docs 폴더 내부에 test.md 파일을 작성한다.
   - md파일은 github markdown 서식을 사용한다.
1. _sidebar.md를 수정한다.
    ```md
      - [test](test.md)
    ```
1. _sidebar.md까지 수정하고나면 Wiki 좌측의 sidebar에 test라는 페이지가 추가된다.
    - 이하 sidebar의 내용은 5단계까지 (# ~ #####) 까지 자동으로 타이틀을 추가하여 준다.
